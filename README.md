🚀 JD Montagens - Sistema de Gerenciamento
Este é um ecossistema digital desenvolvido para otimizar o fluxo de trabalho da JD Montagens, focando na organização de orçamentos, faturamento e controle de status de serviços.

O sistema oferece uma interface moderna, intuitiva e segura para a gestão do dia a dia da empresa.

✨ Funcionalidades Principais
📊 Gestão de Orçamentos: Criação e acompanhamento detalhado de propostas.

💰 Controle de Faturamento: Visualização clara das entradas e pendências financeiras.

📅 Agenda de Serviços: Organização cronológica das montagens programadas.

🔒 Autenticação Segura: Login integrado via Google para garantir que apenas usuários autorizados acessem os dados.

📱 Interface Responsiva: Totalmente adaptado para uso em computadores, tablets e celulares.

🛠️ Tecnologias Utilizadas
Este projeto foi construído com o que há de mais moderno no desenvolvimento web:

React + Vite: Para uma interface rápida e reativa.

TypeScript: Garantindo um código mais robusto e livre de erros.

Firebase: Utilizado para banco de dados em tempo real e autenticação.

Tailwind CSS: Para um design elegante, profissional e personalizado.
